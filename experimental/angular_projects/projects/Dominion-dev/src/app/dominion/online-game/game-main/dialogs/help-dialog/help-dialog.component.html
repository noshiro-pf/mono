サイドバーのボタン

textsms

チャット画面の表示／非表示を切り替えます。

stop

【チャットを開いているときのみ表示】 チャットを自動でスクロールしないようにします。

arrow_downwards

【チャットを開いているときのみ表示】 チャットのスクロールを自動で行います。

description

カードの下の情報を見るボタンの表示／非表示を切り替えます。

tune

設定画面を開きます。

自分のカードエリアのボタン

history

操作ログを見ることができます。

hourglass_full

考え中のときに押してください。 プレイヤーのエリアに hourglass_fullが表示されます。 （もう一度押すか、ゲーム内で何か操作をすると消えます）

sort

手札をソートします。 （自動でソートさせたい場合は、 設定画面で「手札を自動でソート」にチェックを入れてください）

present_to_all

【購入フェーズのみ表示】基本財宝カード（効果がコインのみの財宝カード） をすべてプレイします。

file_download

【自分の手番のみ表示】次のフェーズに移ります。（確認あり）

forward

【自分の手番のみ表示】ターンを終了します。（確認あり）

ゲーム終了時に表示

format_list_numbered

ゲーム結果を表示します。

view_comfy

各プレイヤーの山札の一覧を表示します。

exit_to_app

ゲームを終了し部屋リスト画面に戻ります。
